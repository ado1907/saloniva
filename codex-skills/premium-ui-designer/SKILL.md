# Premium UI Designer

Use this skill to make interfaces feel polished, premium, and commercially credible.

## Visual Direction

- Prefer calm luxury over flashy decoration.
- Use restrained contrast, warm neutrals, deep ink tones, champagne/gold accents, and a single strong brand accent.
- Avoid one-note palettes, heavy gradients, generic blobs, and decorative effects that do not support the product.
- For Saloniva, avoid illustrated women effects unless the user explicitly asks for that style again.
- Prefer premium salon signals: clean product displays, mirrors, treatment rooms, elegant surfaces, appointment flow, and high-end service details.

## Layout

- Build the actual product screen, not a marketing landing page, unless the user asks for marketing.
- Keep business screens dense but breathable.
- Use cards for individual records, modals, and tools, not nested decorative sections.
- Keep important actions close to the workflow.
- Make text fit on mobile and desktop.

## Components

- Use icons for navigation and actions when available.
- Keep border radius modest unless the existing design system says otherwise.
- Keep buttons clear, tappable, and consistent.
- Use consistent card headers, metadata rows, totals, filters, and status pills.
- Include useful empty, success, and warning states.

## Store Readiness

- Make the app screenshot-friendly.
- Keep the first screen immediately understandable.
- Show realistic demo data with strong business value.
- Avoid placeholder visuals that make the product feel unfinished.
