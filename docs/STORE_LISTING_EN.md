# Saloniva - Store Listing EN

## App name
Saloniva

## Short description
Premium appointment, customer, package, session and payment management for salons and beauty businesses.

## Long description
Saloniva is a premium salon management app built for beauty salons, hair studios, nail bars and wellness businesses that want to manage daily operations from one clean workspace across web, Android and iOS.

The app helps salon owners track appointments, customer history, active packages, remaining sessions, staff workload and payment status from a focused business dashboard. The demo environment includes realistic salon data, while the Supabase-based cloud architecture is designed for separated salon data and secure row-level access policies.
