# White-Label Teslim Paketi

## Teslim Amacı

Müşteriye Saloniva altyapısını kendi markasıyla kullanabileceği, satış görüşmesinde net anlatılabilir ve sözleşmede kapsamı çizilebilir şekilde teslim etmek.

## Teslim Kapsamı

- Müşteri markasına uyarlanmış uygulama adı
- Logo ve temel marka renkleri
- Premium giriş / tanıtım ekranı
- Yönetim paneli erişimi
- Randevu, müşteri, paket, ödeme, personel ve rapor ekranları
- Online randevu talep akışı
- Temel kullanıcı rolleri ve yetkilendirme mantığı
- Supabase bulut backend kurulum hazırlığı
- Kısa kullanım eğitimi veya kullanım notları
- Test sonrası teslim kontrol listesi

## Müşteriden İstenecek Bilgiler

- Marka adı
- Logo dosyası
- Ana renkler
- İletişim bilgileri
- Hizmet/fiyat listesi
- Personel listesi
- Şube bilgileri
- Kullanılacak demo / gerçek veri tercihi
- Yayınlanacak platformlar: web, Android, iOS

## Teslim Edilecek Dosyalar / Erişimler

- Kullanım linki veya demo linki
- Yönetim paneli giriş bilgileri
- Kurulum ve kullanım notları
- White-label kapsam özeti
- Destek ve bakım koşulları
- Teklif / sözleşme eki

## Teslim Edilmeyenler

- Kaynak kod
- Çekirdek yazılım mimarisi
- Veritabanı gizli yönetici anahtarları
- Başka müşterilere ait yapılandırmalar
- Saloniva'nın genel ürün hakları

## Teslim Kontrol Listesi

| Kontrol | Durum |
|---|---|
| Marka adı işlendi | Bekliyor |
| Logo işlendi | Bekliyor |
| Renkler uyarlandı | Bekliyor |
| Demo veriler düzenlendi | Bekliyor |
| Yönetim paneli test edildi | Bekliyor |
| Mobil görünüm kontrol edildi | Bekliyor |
| Web görünüm kontrol edildi | Bekliyor |
| Kullanım notları hazırlandı | Bekliyor |
| Teslim görüşmesi yapıldı | Bekliyor |

## Destek Modeli

White-label teslim sonrası destek üç seviyeye ayrılabilir:

- Standart destek: hata düzeltme ve küçük kullanım soruları
- Profesyonel destek: küçük ekran/metin düzenlemeleri ve kurulum yardımı
- Gelişmiş destek: yeni özellik, özel entegrasyon ve rapor geliştirme
