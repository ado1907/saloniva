# Teslim Kabul Kontrol Listesi

Bu liste, Saloniva tesliminde alıcıyla birlikte kontrol edilmek için hazırlanmıştır.

## Marka ve Arayüz

- [ ] Uygulama adı doğru.
- [ ] Logo işlendi.
- [ ] Ana renkler uygulandı.
- [ ] Splash / ikon varlıkları kontrol edildi.
- [ ] Demo ekran dili alıcı markasına uygun.

## Hesap ve Yetki

- [ ] İlk yönetici hesabı oluşturuldu.
- [ ] Salon sahibi rolü atandı.
- [ ] Yönetici/personel rol modeli anlatıldı.
- [ ] Şifre sıfırlama akışı kontrol edildi.

## Veri ve Supabase

- [ ] Supabase URL ve publishable key doğru.
- [ ] `schema.sql` çalıştırıldı.
- [ ] `phase2-auth.sql` çalıştırıldı.
- [ ] RLS politikaları aktif.
- [ ] Salon bazlı veri ayrımı test edildi.
- [ ] İkinci salon testinde veri ayrımı doğrulandı.

## Uygulama Akışları

- [ ] Randevu oluşturma test edildi.
- [ ] Randevu ödeme alma test edildi.
- [ ] Müşteri ekleme test edildi.
- [ ] Müşteri notu test edildi.
- [ ] Paket/seans takibi test edildi.
- [ ] Ödeme kaydı test edildi.
- [ ] Online randevu talebi test edildi.
- [ ] Stok uyarıları kontrol edildi.

## Mobil ve Web

- [ ] Web önizleme açıldı.
- [ ] Android görünüm kontrol edildi.
- [ ] iOS görünüm kontrol edildi veya planlandı.
- [ ] Küçük ekran taşma kontrolü yapıldı.
- [ ] Alt navigasyon test edildi.

## Lisans ve Destek

- [ ] Kaynak kod devri olmadığı kabul edildi.
- [ ] Kullanım lisansı kapsamı kabul edildi.
- [ ] Destek kapsamı kabul edildi.
- [ ] Ek geliştirmelerin ayrı fiyatlanacağı kabul edildi.
- [ ] Mağaza hesap sorumluluğu netleşti.
- [ ] Supabase sahiplik modeli netleşti.
